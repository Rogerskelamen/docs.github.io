# 响应报文的各种内容格式

> 在`content-type`（仅限响应头）之后可以标注上各种内容的格式，来指明可以用哪些格式来渲染页面。

| 名称                   | 相应格式     |
| ---------------------- | :----------- |
| text/plain             | 纯文本       |
| text/html              | html格式解析 |
| text/css               | css文件      |
| application/javascript | js文件       |
| image/jpeg             | 图片文件     |
| application/json       | json数据文件 |
